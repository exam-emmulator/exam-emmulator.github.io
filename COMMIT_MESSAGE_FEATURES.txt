## Example Usage

```json
{
  "id": "certification-exam",
  "name": "AWS Solutions Architect",
  "shuffleQuestions": true,
  "shuffleOptions": true,
  "passingScore": 72,
  "timeLimit": 130,
  "sections": [
    {
      "name": "Domain 1: Design",
      "weight": 30
    }
  ],
  "questions": [
    {
      "question": "Design a highly available architecture...",
      "options": ["A", "B", "C", "D"],

      "section": "Domain 1: Design",

      "difficulty": "medium"

  ]
}
```
